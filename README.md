# MissChief

A modern website concept created for **MissChief Records**, designed to present the label's artists, music, and brand identity through a clean, responsive, and visually engaging experience.

This project focuses on creating an immersive landing page with smooth navigation, bold visuals, and a user-friendly interface while demonstrating modern front-end development practices.

## 🌐 Live Demo

https://emeraldkingg.github.io/Misschief/

## ✨ Features

- Responsive design for desktop and mobile
- Modern landing page layout
- Smooth scrolling navigation
- Artist and music showcase
- Clean typography and visual hierarchy
- Interactive hover effects and transitions
- Optimized user experience

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🎯 Purpose

This project was created to strengthen my front-end development skills by designing and building a modern website for a creative brand. It reflects my approach to combining clean code with thoughtful UI design and responsive development.

- Improving responsive layouts

- Building smoother user interactions
- Writing cleaner and more maintainable code
